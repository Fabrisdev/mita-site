# Mita site
Mita site is a website developed with Next.js to make it easier to configure [Mita's bot](https://github.com/Fabrisdev/mita-bot), a bot made for Mita's Heart community.

## Getting Started
1. Create your .env file  
    ```bash
    cp .env.example .env
    ```
2. Populate all fields
3. Install dependencies
    ```bash
    bun install
    ```
4. Start server
    ```bash
    bun run start # For production mode
    bun run dev # or dev for development mode
    ```

You will be able to see the site at [http://localhost:3000](http://localhost:3000).
